# 払込取扱票 印字ツール

郵便振込用紙（払込取扱票）に金額・通信欄・おところ（住所）・おなまえ（氏名）を印字するブラウザツールです。
`index.html` を開くだけで動作します（インストール・ネット接続不要）。

## 使い方

1. 金額・住所・氏名などを入力
2. 「位置確認用の枠線も印刷する」をONにして普通紙に試し刷りし、実物の用紙と透かして重ねて位置を確認
3. ズレがあれば「印字位置の微調整」で補正（設定はブラウザに自動保存）
4. 枠線をOFFに戻し、払込取扱票をプリンタにセットして本番印刷

印刷ダイアログでは **倍率100%（実際のサイズ）・余白なし** を選択してください。

## 設定の引き継ぎ

「設定を書き出す」で保存した `haraikomi-settings.json` を、別のパソコンで「設定を読み込む」と位置調整を引き継げます。

## 用紙サイズ

180×114mm を前提としています。実物と異なる場合は `index.html` 内の `DEFAULTS` を調整してください。
